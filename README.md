# Game Systems Portfolio

Unity 기반으로 다양한 게임 시스템을 구현한 프로젝트입니다.

---

## 레포지토리 소개
이 레포지토리는 게임의 주요 시스템들의 구현 코드를 정리한 레포지토리입니다.

---

## 사용 기술
- Unity
- C#

---

## 폴더 내용
- Systems: 게임의 주요 기능 시스템
- Managers: 시스템 흐름 및 상태 관리
- Dataform: 데이터 구조 정의
- Dispatchers: 전달받은 ID와 데이터를 기반으로 적절한 기능을 분기 실행하는 디스패처 모음

---

## 선정 포인트
- 반복 작업을 추상화하여 코드의 복잡도를 낮추고 개발 효율을 향상시키는가  
- 게임 진행 및 시스템 간 연계를 보조하는 역할을 수행하는가  
- 다양한 컴포넌트에서 재사용 가능하도록 구조화되어 있는가
